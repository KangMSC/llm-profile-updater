${existingProfileJson}

마지막 업데이트 이후 이 캐릭터에게 발생한 새로운 이벤트는 다음과 같습니다:

${formattedEvents}

**전역 규칙**
${globalRulesString}

${instructionsString}
${customKeysString}

새로운 이벤트와 제공된 모든 지침(전역 및 특정)을 바탕으로 JSON 프로필의 모든 키 값을 업데이트하세요.
특정 필드에 대한 지침이 있는 경우, 해당 지침에 따라 내용을 생성하세요.
값을 변경할 필요가 없는 경우, 원본을 유지하세요.
당신의 임무는 완전하고 업데이트된 JSON 객체를 출력하는 것이며, 다른 어떤 것도 출력해서는 안 됩니다.

항상 한국어로 응답하세요.

- ${requiredKeys}
